# ⚡ 快速开始指南

5分钟上手考种平台 v2.0！

---

## 📦 第一步：安装环境

### 方式1：使用 Conda（推荐）

```bash
conda env create -f environment.yaml
conda activate SeedTest
```

### 方式2：使用 pip

```bash
pip install -r requirements.txt
```

---

## 🤖 第二步：准备模型

1. 将 YOLOv8 模型文件放入 `models/` 文件夹：

```
models/
├── soybean_640_n.pt       # 你的模型
├── wheat_640.pt           # 你的模型
└── model_mapping.csv      # 映射文件
```

2. 编辑 `models/model_mapping.csv`：

```csv
soybean_640_n.pt,大豆-普通检测-640
wheat_640.pt,小麦-普通检测-640
```

💡 **提示：** 如果没有模型，程序仍可启动，只是无法进行考种分析

---

## 🚀 第三步：启动程序

### Windows

双击 `启动.bat` 或运行：
```bash
python start.py
```

### Linux/Mac

```bash
chmod +x 启动.sh
./启动.sh
```

或直接：
```bash
python start.py
```

---

## 🎯 第四步：快速测试

### 测试1：图像采集（无需模型）

1. 主页点击 **"图像采集"**
2. 点击 **"扫描相机"** → 选择相机 → **"打开相机"**
3. 点击 **"开始预览"**
4. 点击 **"拍照"** 测试

✅ 如果能看到相机画面并拍照，说明相机工作正常

### 测试2：ROI设置（无需模型）

1. 主页点击 **"设置"**
2. **"相机与ROI"** 标签页
3. 打开相机 → **"开始预览"**
4. 用鼠标在预览区域**拖拽绘制矩形**
5. 点击 **"应用ROI"** → **"保存设置"**

✅ 如果能框选ROI，说明设置功能正常

### 测试3：考种分析（需要模型）

1. 主页点击 **"考种"**
2. 在弹出的对话框中**选择模型** → **"确定"**
3. 打开相机和天平（可选）
4. **"开始预览"** → **"处理图像"**
5. 调整参数 → **"保存数据"**

✅ 如果能完成一次考种，说明系统完全正常

## 🔧 常见问题速查

### ❌ 未检测到相机

**解决：**
- 检查USB连接
- 重新扫描相机
- 确认海康SDK已安装

### ❌ 未找到模型

**解决：**
- 检查 `models/` 文件夹是否有 `.pt` 文件
- 检查 `models/model_mapping.csv` 是否存在
- 重启程序

### ❌ 天平数据异常

**解决：**
- 点击 "清零" 按钮
- 重新插拔USB
- 重新扫描天平

### ❌ 程序启动失败

**解决：**
```bash
# 运行测试脚本
python test_new_architecture.py

# 查看哪个模块有问题
# 根据提示修复
```

## 💡 小技巧

### 技巧1：快速切换模型

```
主页 → 考种 → 选择不同模型 → 即可切换
```
无需重启程序！

### 技巧2：保存常用配置

```
主页 → 设置 → 调整参数 → 保存设置
```
下次启动自动加载！

### 技巧3：批量导出数据

```
考种 → 数据管理 → 导出Excel
```
所有记录一次性导出！
